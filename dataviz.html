Who Makes the Best Board Games?

An Analysis of Board Game Designers and Publishers


Data and Goals

    Dataset Selection:
    • Data: Board Game data scraped from BoardGameGeek
    • Data Source: Kaggle dataset by Bojan Drango
    • Data Format: CSV
    • Data Dimensions:
      • Game ID
      • Game Name
      • Game Type
      • Designer
      • Publisher
      • Minimum Number of Players
      • Maximum Number of Players
      • Minimum Play Time
      • Maximum Play Time
      • Minimum Age
      • Number of Owners

    Goals:
    • Rank Board Game Designers:
      • Create plot of Designers sorted by Number of Games Designed and Average Game Rating.
    • Rank Board Game Publishers:
      • Create plot of Publishers sorted by Number of Games Published and Average Game Rating.
    • Determine Best Designer/Publisher Partnerships:
      • Combined the two previous datasets and create a plot to show the highest ranted Designer/Publisher pairings.
